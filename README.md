Class attendance checker

Class attendance checker is a Python script that helps in automating the task of checking the attendance of students by providing features such as searching a student, show present or absent students in a separated list, and uploading the lists to a file. This script was made for teachers or professors. The script includes a friendly terminal user interface to enhance user experience and organized menu.
